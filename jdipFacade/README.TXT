===================
README.TXT for jDip
===================

jdipFacade is a developer's API for the Jdip diplomacy adjudication package.
More information may be found at http://jdip.sf.net/


Requirements
============
  * Same as Jdip.
  * jdipFacade will not work, or even compile without a working jdip.jar. You
    must download and install Jdip to use jdipFacade.
